< head > < meta charset = " utf-8 " > < meta content = ' width = device-width, initial-scale = 1 ' name = ' viewport ' > < body > Meningkatkan kinerja smartphone Pastikan terlebih dahulu penyimpanan eksternal maupun internal smartphone anda mencukupi, sebab jika ruang penyimpanan penuh maka seluruh aplikasi yang aktif akan bekerja tidak maximal. Salah satu hal yang bisa anda lakukan ialah:
  • Menghapus data cache Data cache adalah file sampah yang berasal dari hasil jalannya aplikasi pada smartphone, contohnya seperti setelan pada aplikasi android dan file tambahan yang sudah tidak digunakan seperti ssisa dari penelusuran pada browser dan lainnya. Untuk menghapus seluruh file cache anda bisa menggunakan aplikasi tambahan clear chache yang bisa didownload pada google play, silakan anda bersihkan sampai habis file chache tersebut. Anda tidak perlu takut kehilangan data penting karena yang akan dihapus hanyalah file sampah yang tersimpan pada setiap aplikasi. jika smartphone Android yang dipakai telah memiliki aplikasi penghapus cache bawaan smartphone sebaiknya gunakan itu saja, jadi tidak perlu mendownload aplikasi menghapus file cache lainnya di playstore.
  • Hapus aplikasi yang tidak terpakai Apabila smartphone android mu kerap mengalami masalah force close seperti tiba-tiba keluar dan muncul tulisan sayangnya telah berhenti, biasanya disebabkan karena banyaknya aplikasi yang sedang bekerja sehingga untuk memfokuskan kepada satu aplikasi menjadi berat karena tenaga RAM sudah dibagi ke beberapa aplikasi lainnya yang sedang berjalan. solusi untuk mengatasi masalah seperti ini sebaiknya aplikasi yang tidak dipakai dihapus, meskipun jarang dibuka karena beberapa aplikasi yang bersifat online akan bekerja walaupun tidak dibuka contohnya seperti munculnya notifikasi padahal kita tidak membukanya.
Tambah penyimpanan external Masalah smartphone android menjadi lemot dan sering muncul notifikasi tulisan sayangnya aplikasi telah berhenti disebabkan karena memori penyimpanan sudah penuh. Hal ini menyebabkan aplikasi yang berjalan tidak bisa menyimpan data, jika diteruskan seperti itu lama kelamaan smartphone anda akan macet total tidak bisa di operasikan (botloop) sebelum kejadian ini benar-benar terjadi pada smartphone anda sebaiknya tambah memori eksternal untuk penyimpanan data yang sekiranya penting dan jarang dibuka atau bisa menggunakan penyimpanan cloud seperti Google Drive dan penyedia penyimpanan gratis lainnya. Jika smartphone milik anda tidak menyediakan slot external memory sebaiknya pindahkaan data penting ke media lainnya agar memiliki ruang penyimpanan yang cukup untuk beroperasi. Restart Smartphone guna mengembalikan kondisi seperti awal Saat mengalami kendala seperti force close atau keluar secara tiba-tiba dan sering muncul notifikasi tulisan sayangnya telah berhenti cobalah untuk me-restart hp anda guna mengembalikan kondisi yang lebih baik. Tahap ini dilakukan juga berguna menghentikan seluruh aktivitas yang memberatkan smartphone seperti lemot dan dan macet-macet. Uninstall aplikasi yang memberatkan smartphone Banyak pengguna smartphone memanfaatkannya untuk aktifitas hiburan seperti game. Namun perlu di perhatikan tidak semua aplikasi game suport terhadap smartphone yang anda punya. Sebaiknya sebelum memasang aplikasi berat perhatikan terlebih dahulu spesifikasi smartphone yang di butuhkan supaya kedepannya tidak ada kendala, pastikan ruang penyimpanan smartphone mencukupi.
Pastikan daya ponsel cukup Jika pernah mengalami masalah force close atau macet-macet saat menjalankan aplikasi perhatikan terlebih dahulu daya pada smartphone anda, karena daya yang kurang akan membuat kinerja smartphone menurun contohnya saja pada aplikasi game. Game membutuhkan grafik yang bagus untuk menampilkan gambar yang baik belum lagi jika game tersebut online tentunya akan membutuhkan banyak daya agar performa game lebih baik. Nah jika daya pada smartphone kurang maka akibat yang terjadi ialah macet-macet force close dan kadang terhenti lalu memunculkan tulisan sayangnya aplikasi telah berhenti.
